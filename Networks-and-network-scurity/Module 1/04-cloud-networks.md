# Cloud Computing and Cloud Networks

## What is Cloud Computing?

**Cloud computing** is the use of remote servers, applications, storage, and network services hosted on the Internet instead of local physical infrastructure.

Instead of owning and maintaining hardware, organizations access cloud resources whenever they need them.

---

# Why Organizations Use Cloud Computing

Cloud computing helps organizations:

- Reduce infrastructure costs
- Minimize hardware maintenance
- Scale resources as needed
- Access services from anywhere
- Simplify network management
- Improve flexibility and availability

---

# Traditional (On-Premises) Network

### Definition

An **on-premises network** stores servers, applications, and networking equipment at the organization's physical location.

### Characteristics

- Organization owns the hardware
- IT team manages infrastructure
- Higher maintenance costs
- Greater physical control over systems

---

# Cloud Network

### Definition

A **cloud network** is a collection of remote servers and computing resources hosted in a **data center** and accessed over the Internet.

Because the infrastructure is hosted remotely, it is said to be **"in the cloud."**

### Characteristics

- Remote servers
- Internet-based access
- Vendor-managed infrastructure
- Scalable resources
- Accessible from any location

---

# On-Premises vs Cloud Network

| Feature | On-Premises | Cloud Network |
|---------|-------------|---------------|
| Hardware Location | Organization's site | Remote data center |
| Maintenance | Organization | Cloud provider |
| Cost | Higher upfront investment | Pay-as-you-use |
| Scalability | Limited by hardware | Highly scalable |
| Accessibility | Usually local | Accessible via the Internet |

---

# Cloud Service Providers (CSPs)

### Definition

A **Cloud Service Provider (CSP)** is a company that delivers cloud computing services over the Internet.

### Services Offered

- Cloud storage
- Computing power
- Applications
- Databases
- Networking services
- Analytics
- Backup and recovery

---

# Common Cloud Services

Cloud providers commonly offer:

### Storage

- Store files and databases remotely

### Computing Power

- Virtual machines
- Processing resources
- On-demand servers

### Applications

- Business software
- Web applications
- Collaboration tools

### Analytics

- Website traffic analysis
- Business intelligence
- Performance monitoring

---

# Benefits of Cloud Computing

- Lower operational costs
- Reduced hardware requirements
- High availability
- Flexible resource allocation
- Easy scalability
- Remote accessibility
- Automatic maintenance and updates
- Faster deployment of services

---

# Cloud Security

### Definition

**Cloud security** is the practice of protecting cloud-based systems, applications, data, and infrastructure from cyber threats.

As organizations increasingly move to the cloud, securing these environments becomes a critical responsibility for security professionals.

---

# Identity-Based Security

Cloud environments place greater emphasis on **identity-based security**.

Security teams verify:

- **Who** is requesting access (identity)
- **Where** the request is coming from (device, network, or location)

This complements traditional network security, which primarily focuses on securing network traffic.

---

# Role of Security Analysts

Security analysts help secure cloud environments by:

- Monitoring cloud activity
- Verifying user identities
- Detecting suspicious behavior
- Protecting cloud resources
- Investigating security incidents
- Enforcing access controls

---

# Traditional Network vs Cloud Network

| Traditional Network | Cloud Network |
|---------------------|---------------|
| Physical servers on-site | Remote servers in data centers |
| Organization manages hardware | Cloud provider manages infrastructure |
| Higher maintenance effort | Reduced maintenance |
| Fixed resources | On-demand resources |
| Limited remote access | Accessible from anywhere |

---

# Key Takeaways

- **Cloud computing** uses remote servers, applications, storage, and networking services hosted on the Internet instead of local infrastructure.
- A **cloud network** consists of remotely hosted resources located in data centers and accessed over the Internet.
- Organizations adopt cloud computing to **reduce costs, improve scalability, simplify operations, and increase flexibility**.
- **Cloud Service Providers (CSPs)** deliver services such as storage, computing power, networking, applications, and analytics on a **pay-as-you-use** model.
- **Cloud security** focuses on protecting cloud-based systems, data, and services from cyber threats.
- Modern cloud environments rely heavily on **identity-based security**, verifying both **who** is accessing resources and **where** the access request originates.
- As more organizations migrate to cloud environments, cloud security has become an essential part of modern cybersecurity.
