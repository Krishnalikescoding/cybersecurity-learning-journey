# Module Key Terms

| **Term** | **Definition** |
|----------|----------------|
| **Bandwidth** | Maximum data transmission capacity of a network, measured in bits per second (bps). |
| **Cloud Computing** | Using remote servers, applications, and network services over the Internet instead of local devices. |
| **Cloud Network** | A collection of remote servers and resources hosted in data centers and accessed via the Internet. |
| **Data Packet** | The basic unit of information transmitted between devices on a network. |
| **Hub** | A network device that broadcasts data to every connected device. |
| **Internet Protocol (IP)** | Protocol used for addressing and routing data packets across networks. |
| **IP Address** | A unique logical address that identifies a device on a network or the Internet. |
| **Local Area Network (LAN)** | A network covering a small area, such as a home, school, or office. |
| **Media Access Control (MAC) Address** | A unique hardware identifier assigned to a network interface. |
| **Modem** | A device that connects a local network to an Internet Service Provider (ISP). |
| **Network** | A group of connected devices that communicate and share resources. |
| **OSI Model** | A 7-layer conceptual model describing how network communication occurs. |
| **Packet Sniffing** | The process of capturing and inspecting network packets. |
| **Port** | A software-based communication endpoint used to identify network services and applications. |
| **Router** | A device that connects multiple networks and forwards packets using IP addresses. |
| **Speed** | The rate at which data is transmitted or received, measured in bits per second (bps). |
| **Switch** | A device that forwards data only to the intended device using MAC addresses. |
| **TCP/IP Model** | A 4-layer framework describing how data is organized and transmitted across networks. |
| **Transmission Control Protocol (TCP)** | A reliable, connection-oriented protocol that ensures accurate data delivery. |
| **User Datagram Protocol (UDP)** | A fast, connectionless protocol that does not guarantee delivery. |
| **Wide Area Network (WAN)** | A network covering a large geographic area, such as a city, state, or country. |

---

# Quick Revision

- **LAN** → Small network (Home, Office)
- **WAN** → Large network (City, Country)
- **Hub** → Broadcasts to all devices
- **Switch** → Sends data only to the correct device
- **Router** → Connects different networks
- **Modem** → Connects the network to the Internet
- **TCP** → Reliable, connection-oriented
- **UDP** → Fast, connectionless
- **IP Address** → Logical address of a device
- **MAC Address** → Physical hardware address
- **Bandwidth** → Maximum network capacity
- **Speed** → Actual data transfer rate
- **Packet Sniffing** → Capturing and analyzing network packets
- **Cloud Computing** → Using remote computing resources over the Internet
- **TCP/IP Model** → 4-layer networking model
- **OSI Model** → 7-layer networking reference model
