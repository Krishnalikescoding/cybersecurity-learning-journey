# Wi-Fi Security Protocols

## IEEE 802.11 (Wi-Fi)

### Definition

**IEEE 802.11**, commonly known as **Wi-Fi**, is a family of standards that defines how devices communicate over **Wireless Local Area Networks (WLANs)**.

- **IEEE** stands for **Institute of Electrical and Electronics Engineers**.
- **802.11** is the set of protocols used for wireless networking.

---

# Evolution of Wi-Fi Security

As wireless networking became more common, stronger security protocols were introduced to protect wireless communications.

### WPA (Wi-Fi Protected Access)

- Introduced in **2004**
- Replaced the insecure **WEP** protocol
- Improved wireless authentication and encryption
- Increased the security of Wi-Fi networks

---

### WPA2

- Successor to WPA
- Uses **AES (Advanced Encryption Standard)** encryption
- Provides stronger security and better protection against attacks
- Became the standard for most wireless networks

---

### WPA3

- Latest Wi-Fi security standard
- Offers stronger encryption and authentication
- Better protection against password-guessing (brute-force) attacks
- Improves security for both personal and enterprise networks

---

# WPA Versions Comparison

| Protocol | Features |
|----------|----------|
| **WPA** | Improved security over WEP with stronger encryption and authentication |
| **WPA2** | Uses AES encryption and provides stronger wireless security |
| **WPA3** | Strongest security, enhanced encryption, and improved protection against attacks |

---

# Importance for Security Analysts

Security analysts are responsible for ensuring that:

- Wireless networks use modern security protocols.
- Older, insecure protocols are replaced.
- Wi-Fi networks are configured securely.
- Unauthorized access to wireless networks is prevented.

---

# Key Takeaways

- **IEEE 802.11 (Wi-Fi)** is the standard for wireless LAN communication.
- **IEEE** develops and maintains Wi-Fi standards.
- **WPA** was introduced in **2004** to improve wireless security.
- **WPA2** strengthened security by using **AES encryption**.
- **WPA3** provides the strongest wireless protection with improved encryption and authentication.
- Using the latest Wi-Fi security protocols helps protect wireless networks from unauthorized access and cyberattacks.
