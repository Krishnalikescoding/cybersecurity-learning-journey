# Module 4: Terms & Definitions

- **Antivirus Software:** Software used to prevent, detect, and remove malware and viruses.
- **Database:** An organized collection of information or data.
- **Data Point:** A single piece of information stored in a database.
- **Intrusion Detection System (IDS):** Monitors system or network activity and alerts on possible intrusions.
- **Linux:** An open-source operating system.
- **Log:** A record of events that occur within an organization's systems.
- **Network Protocol Analyzer (Packet Sniffer):** A tool that captures and analyzes network traffic.
- **Order of Volatility (OoV):** The sequence for collecting digital evidence, starting with the most volatile data.
- **Programming:** The process of creating instructions that tell a computer how to perform tasks.
- **Protecting & Preserving Evidence:** Properly handling fragile and volatile digital evidence to maintain its integrity.
- **Security Information and Event Management (SIEM):** A tool that collects and analyzes log data to monitor security events and detect threats.
- **SQL (Structured Query Language):** A language used to create, manage, and query databases.
